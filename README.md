# 🏨 Hotel Website Design
## 📌 Project Overview

This project is a hotel website designed using Canva, following modern web design layout principles and usability guidelines.

The goal of this project was to create a visually appealing, user-friendly, and responsive hotel landing page that reflects professional UI/UX standards.

## 🔗 Live Website:
https://myhoteldesignko.my.canva.site/

## 🎯 Objectives

* Apply web design layout rules (grid structure, spacing, visual hierarchy)
* Follow modern UI/UX best practices
* Create a clean and professional hotel website concept
* Maintain consistent typography and color scheme
